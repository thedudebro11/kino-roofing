# Kino Roofing — AI Image Generation Prompts

## Visual Style Guide

All images should feel like professional commercial photography for a local roofing company, not generic stock photos.

- **Location:** Sonoran Desert / Tucson, AZ — saguaro cacti, desert landscaping, no green lawns
- **Architecture:** Adobe, stucco, or Southwestern ranch-style homes; earth tones (tan, cream, terracotta)
- **Sky:** Deep cobalt blue desert sky; intense desert sunlight with sharp shadows
- **Color palette:** Warm oranges, tan, terracotta, charcoal gray roofing, bright white TPO
- **Mood:** Professional, competent, trustworthy — not staged or overly glossy
- **No text, logos, or watermarks** in any generated image

## Logo

The logo (`public/images/logo.png`) should be created in a **design tool**, not AI-generated:

- **Canva (free):** Use a construction/home icon template → apply orange `#ea580c` → set name "Kino Roofing" in a clean bold sans-serif
- **Figma (free):** Custom vector design
- **Icon ideas:** Simple house silhouette, rooftop triangle, or combined K + roof shape in orange

**Export size:** 300 × 80 px, PNG with transparent background. Also export a white version for use on the dark footer.

## Recommended AI Image Tools

| Tool | Best for | Notes |
|------|---------|-------|
| **Midjourney** | Photorealistic scenes, people | Append `--ar 3:2` or `--ar 16:9` for aspect ratio |
| **DALL-E 3** (ChatGPT) | Quick generation, follows prompts literally | Use in ChatGPT or via API |
| **Adobe Firefly** | Commercial-safe images | Good for people photos |
| **Stable Diffusion** | Full control, local generation | Requires setup |

After generating, use **Squoosh** (squoosh.app) to convert to WebP and compress before saving.
